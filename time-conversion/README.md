# Time Conversion

https://www.hackerrank.com/challenges/time-conversion/problem

Submitted in 2023.

## Problem

Given a time in 12-hour AM/PM format, convert it to military (24-hour) time.

### Example

```python
s = '12:01:00PM'
```

Return '12:01:00'.

```python
s = '12:01:00AM'
```

Return '00:01:00'.
